Shoppy

Web-based shared shopping list

made with JS Vue Firebase

Shoppy is a shopping list you can share with anyone. As a progressive web app hooked up to a real-time database, your shopping lists will be synchronized between users effortlessly.

I built Shoppy to familiarize myself with Vue.js and Firebase. On the frontend, I use Vue, as well as Quasar Framework as a Material design component library. Firebase is used for user authentication, its real-time database, hosting, and several serverless cloud functions. Firebase allows me to have a complete backend without the need to set up any server by myself.
