Asset Finder

A Unity asset

made with C# Unity

Asset Finder is a tiny Unity editor utility which helps you quickly find the assets you are looking for, without clicking through the Asset Browser. You can use the tool using only the keyboard, saving yourself the hassle of clicking through the Project view.

Asset Finder is the third in my series of Unity editor plugins which I released, and it is by far the best. The inspiration to make this asset came from Unreal Engine 4, which contains a similar feature to search through source files. While Unity has a decent search feature, using it can be cumbersome. As with many other parts of Unity, lots of clicking around is required. Ideally, you would want to keep your hands on the keyboard to interrupt your workflow as minimally as possible. Using this plugin, you can do just that!
